# Excel VBA Projects

A professional portfolio of Excel VBA solutions built to demonstrate advanced spreadsheet automation, reporting workflows, business controls, and productivity-driven tools.

This repository brings together a structured collection of Excel VBA projects designed to reflect practical business use cases across finance, audit, compliance, administration, operations, HR, and reporting environments.

Rather than focusing on simple macros alone, these projects aim to show how Excel can be transformed into lightweight business applications through structured VBA logic, automation, and clean workbook design.

---

## Repository Purpose

The purpose of this repository is to build a strong Excel VBA portfolio that demonstrates:

- practical VBA development
- automation of repetitive tasks
- reporting and dashboard preparation
- business process improvement
- data validation and control logic
- structured spreadsheet-based tools
- workflow support for real operational environments

This repository is intended for portfolio presentation, continuous learning, and showcasing real-world automation capabilities in Excel.

---

## Project Portfolio

### 1. Financial Audit & Compliance Tracker
A professional Excel VBA tool for recording audit findings, tracking remediation actions, monitoring overdue items, refreshing dashboards, and exporting executive summaries.

### 2. Automated Expense Tracker
An Excel workbook with VBA automation for recording expenses, validating entries, classifying categories, and generating summaries.

### 3. Invoice Generator
A VBA-driven invoice preparation tool for generating formatted invoices automatically from structured input data.

### 4. Attendance Tracker
A workbook for tracking attendance records, calculating summaries, and preparing monthly status reports.

### 5. Dashboard Automation Tool
A reporting support tool that refreshes dashboard inputs, updates KPIs, and prepares management-ready reporting views.

### 6. Data Cleaning Tool
An Excel VBA utility for formatting datasets, removing duplicates, validating entries, and preparing cleaner reporting outputs.

### 7. Employee Record Manager
A structured workbook for maintaining employee records with automation logic, controlled fields, and organized data views.

### 8. Report Export Tool
A project designed to export filtered reports automatically to PDF, separate sheets, or structured output files.

### 9. Inventory Workbook
A stock tracking workbook that manages inventory records, movement, balances, and automated calculations.

### 10. Mail Merge Helper
A VBA-based helper for preparing Excel-driven communication workflows and document-based mailing support.

### 11. Payroll Processing Tool
A payroll workbook for automating salary calculations, deductions, allowances, and monthly payroll summaries.

### 12. Leave Management Tracker
A tracker for managing leave balances, employee requests, approval status, and leave history records.

### 13. Project Action Tracker
A task and action monitoring tool for assigning owners, managing due dates, tracking progress, and highlighting overdue items.

### 14. Vendor Payment Tracker
A financial operations tool for monitoring vendor invoices, payment due dates, settlement status, and outstanding balances.

### 15. Budget Monitoring Tool
A workbook for tracking budget allocation, actual expenditure, variance analysis, and department-level financial summaries.

### 16. Document Control Register
A structured document register for managing document numbers, revisions, issue status, submission dates, and control workflows.

### 17. Procurement Tracker
A procurement workflow tool for tracking requests, quotations, approvals, purchase orders, and delivery status.

### 18. Risk Register Tool
A risk tracking workbook used to record risks, evaluate likelihood and impact, assign ownership, and monitor mitigation actions.

### 19. Asset Tracking Workbook
A solution for managing company assets, locations, assigned custodians, operational status, and maintenance records.

### 20. Meeting Minutes & Action Log
A structured workbook for recording meeting decisions, action items, assigned owners, deadlines, and follow-up status.
